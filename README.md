# Landing Page Caio Batista

## Jeito mais facil

1. Instale o Node.js LTS: https://nodejs.org/
2. Dê duplo clique em `abrir-site.bat`
3. Abra `http://localhost:3000`

## Scripts prontos

- `abrir-site.bat`: instala dependências se faltar, abre o navegador e inicia o projeto
- `rodar-site.bat`: inicia o projeto local rapidamente
- `instalar-site.bat`: instala as dependências manualmente
- `gerar-producao.bat`: gera o build de produção

## Se preferir pelo terminal
